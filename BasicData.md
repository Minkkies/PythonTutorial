### ‡∏Ñ‡∏∏‡∏ì‡∏™‡∏°‡∏ö‡∏±‡∏ï‡∏¥‡∏´‡∏•‡∏±‡∏Å‡∏Ç‡∏≠‡∏á String

1. **‡∏°‡∏µ‡∏•‡∏≥‡∏î‡∏±‡∏ö (Ordered)**: ‡∏≠‡∏±‡∏Å‡∏Ç‡∏£‡∏∞‡πÅ‡∏ï‡πà‡∏•‡∏∞‡∏ï‡∏±‡∏ß‡∏°‡∏µ‡∏ï‡∏≥‡πÅ‡∏´‡∏ô‡πà‡∏á (index) ‡∏ó‡∏µ‡πà‡πÅ‡∏ô‡πà‡∏ô‡∏≠‡∏ô
   - Index ‡∏ö‡∏ß‡∏Å: ‡∏ô‡∏±‡∏ö‡∏à‡∏≤‡∏Å‡∏ã‡πâ‡∏≤‡∏¢‡πÑ‡∏õ‡∏Ç‡∏ß‡∏≤ ‡πÄ‡∏£‡∏¥‡πà‡∏°‡∏ó‡∏µ‡πà 0
   - Index ‡∏•‡∏ö: ‡∏ô‡∏±‡∏ö‡∏à‡∏≤‡∏Å‡∏Ç‡∏ß‡∏≤‡∏°‡∏≤‡∏ã‡πâ‡∏≤‡∏¢ ‡πÄ‡∏£‡∏¥‡πà‡∏°‡∏ó‡∏µ‡πà -1
2. **‡πÑ‡∏°‡πà‡∏™‡∏≤‡∏°‡∏≤‡∏£‡∏ñ‡πÅ‡∏Å‡πâ‡πÑ‡∏Ç‡πÑ‡∏î‡πâ (Immutable)**: ‡πÑ‡∏°‡πà‡∏™‡∏≤‡∏°‡∏≤‡∏£‡∏ñ‡πÄ‡∏õ‡∏•‡∏µ‡πà‡∏¢‡∏ô‡πÅ‡∏õ‡∏•‡∏á‡∏≠‡∏±‡∏Å‡∏Ç‡∏£‡∏∞‡πÉ‡∏ô‡∏ï‡∏≥‡πÅ‡∏´‡∏ô‡πà‡∏á‡πÄ‡∏î‡∏¥‡∏°‡πÑ‡∏î‡πâ
3. **‡πÄ‡∏õ‡πá‡∏ô‡∏•‡∏≥‡∏î‡∏±‡∏ö (Sequence)**: ‡∏™‡∏≤‡∏°‡∏≤‡∏£‡∏ñ‡∏ß‡∏ô‡∏ã‡πâ‡∏≥ (iterate) ‡πÅ‡∏•‡∏∞‡πÉ‡∏ä‡πâ slicing ‡πÑ‡∏î‡πâ

### ‡∏Å‡∏≤‡∏£‡πÄ‡∏Ç‡πâ‡∏≤‡∏ñ‡∏∂‡∏á‡∏≠‡∏±‡∏Å‡∏Ç‡∏£‡∏∞ (Indexing & Slicing)
```python

# Slicing [start:stop:step]

print(s[:3])     # 'Pyt' (‡∏ï‡∏±‡πâ‡∏á‡πÅ‡∏ï‡πà‡∏ï‡πâ‡∏ô‡∏ñ‡∏∂‡∏á index 2)
print(s[3:])     # 'hon' (‡∏à‡∏≤‡∏Å index 3 ‡∏ñ‡∏∂‡∏á‡∏ó‡πâ‡∏≤‡∏¢)
print(s[::2])    # 'Pto' (‡∏ó‡∏∏‡∏Å ‡πÜ 2 ‡∏ï‡∏±‡∏ß)
print(s[::-1])   # 'nohtyP' (‡∏Å‡∏•‡∏±‡∏ö‡∏î‡πâ‡∏≤‡∏ô)

# ‡πÑ‡∏°‡πà‡∏™‡∏≤‡∏°‡∏≤‡∏£‡∏ñ‡πÅ‡∏Å‡πâ‡πÑ‡∏Ç‡πÑ‡∏î‡πâ
# s[0] = 'J'     # ‚ùå TypeError: 'str' object does not support item assignment
```

- ‡∏ñ‡πâ‡∏≤‡πÄ‡∏£‡∏µ‡∏¢‡∏Å Index ‡πÄ‡∏Å‡∏¥‡∏ô‡∏à‡∏≥‡∏ô‡∏ß‡∏ô‡∏ó‡∏µ‡πà‡∏°‡∏µ‡∏à‡∏∞‡πÄ‡∏Å‡∏¥‡∏î Error (IndexError)
- ‡πÑ‡∏°‡πà‡∏™‡∏≤‡∏°‡∏≤‡∏£‡∏ñ‡πÉ‡∏ä‡πâIndex ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡πÅ‡∏Å‡πâ‡πÑ‡∏Ç‡∏Ñ‡πà‡∏≤‡πÑ‡∏î‡πâ‡πÄ‡∏ä‡πà‡∏ô word[0] = 'J' ‡∏à‡∏∞ error

**‡∏Å‡∏≤‡∏£‡∏à‡∏±‡∏î‡∏£‡∏π‡∏õ‡πÅ‡∏ö‡∏ö (Formatting)**

### String Operations (‡∏Å‡∏≤‡∏£‡∏î‡∏≥‡πÄ‡∏ô‡∏¥‡∏ô‡∏Å‡∏≤‡∏£)

```python
# ‡∏Å‡∏≤‡∏£‡∏ï‡πà‡∏≠‡∏™‡∏ï‡∏£‡∏¥‡∏á (Concatenation)

# ‡∏Ñ‡∏ß‡∏≤‡∏°‡∏¢‡∏≤‡∏ß

    print(char)            # P, y, t, h, o, n

# List comprehension ‡∏Å‡∏±‡∏ö string
```

## 2. List (‡∏•‡∏¥‡∏™‡∏ï‡πå)

> **List** ‡∏Ñ‡∏∑‡∏≠‡πÇ‡∏Ñ‡∏£‡∏á‡∏™‡∏£‡πâ‡∏≤‡∏á‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏ó‡∏µ‡πà‡πÉ‡∏ä‡πâ‡πÄ‡∏Å‡πá‡∏ö‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏´‡∏•‡∏≤‡∏¢‡∏£‡∏≤‡∏¢‡∏Å‡∏≤‡∏£‡πÉ‡∏ô‡∏ï‡∏±‡∏ß‡πÅ‡∏õ‡∏£‡πÄ‡∏î‡∏µ‡∏¢‡∏ß ‡πÄ‡∏õ‡πá‡∏ô **mutable** (‡πÅ‡∏Å‡πâ‡πÑ‡∏Ç‡πÑ‡∏î‡πâ) ‡πÅ‡∏•‡∏∞‡πÄ‡∏õ‡πá‡∏ô sequence ‡∏ó‡∏µ‡πà‡πÄ‡∏£‡∏µ‡∏¢‡∏á‡∏•‡∏≥‡∏î‡∏±‡∏ö

### ‡∏Ñ‡∏∏‡∏ì‡∏™‡∏°‡∏ö‡∏±‡∏ï‡∏¥‡∏´‡∏•‡∏±‡∏Å‡∏Ç‡∏≠‡∏á List

1. **‡∏°‡∏µ‡∏•‡∏≥‡∏î‡∏±‡∏ö (Ordered)**: ‡∏™‡∏°‡∏≤‡∏ä‡∏¥‡∏Å‡∏°‡∏µ‡∏ï‡∏≥‡πÅ‡∏´‡∏ô‡πà‡∏á index ‡∏ó‡∏µ‡πà‡πÅ‡∏ô‡πà‡∏ô‡∏≠‡∏ô
2. **‡πÅ‡∏Å‡πâ‡πÑ‡∏Ç‡πÑ‡∏î‡πâ (Mutable)**: ‡∏™‡∏≤‡∏°‡∏≤‡∏£‡∏ñ‡πÄ‡∏û‡∏¥‡πà‡∏° ‡∏•‡∏ö ‡πÅ‡∏Å‡πâ‡πÑ‡∏Ç‡∏™‡∏°‡∏≤‡∏ä‡∏¥‡∏Å‡πÑ‡∏î‡πâ‡∏´‡∏•‡∏±‡∏á‡∏™‡∏£‡πâ‡∏≤‡∏á
3. **‡∏¢‡∏≠‡∏°‡∏£‡∏±‡∏ö‡∏Ñ‡πà‡∏≤‡∏ã‡πâ‡∏≥ (Allows Duplicates)**: ‡∏™‡∏≤‡∏°‡∏≤‡∏£‡∏ñ‡∏°‡∏µ‡∏Ñ‡πà‡∏≤‡∏ã‡πâ‡∏≥‡∏Å‡∏±‡∏ô‡πÑ‡∏î‡πâ
4. **‡πÄ‡∏Å‡πá‡∏ö‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏´‡∏•‡∏≤‡∏Å‡∏´‡∏•‡∏≤‡∏¢‡∏ä‡∏ô‡∏¥‡∏î (Heterogeneous)**: ‡πÄ‡∏Å‡πá‡∏ö‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏ï‡πà‡∏≤‡∏á‡∏ä‡∏ô‡∏¥‡∏î‡∏Å‡∏±‡∏ô‡πÑ‡∏î‡πâ

### ‡∏Å‡∏≤‡∏£‡∏™‡∏£‡πâ‡∏≤‡∏á List

```python
# ‡πÉ‡∏ä‡πâ square brackets []
numbers = [1, 2, 3, 4, 5]
```

### Set Operations (‡∏Å‡∏≤‡∏£‡∏î‡∏≥‡πÄ‡∏ô‡∏¥‡∏ô‡∏Å‡∏≤‡∏£‡∏ó‡∏≤‡∏á‡∏Ñ‡∏ì‡∏¥‡∏ï‡∏®‡∏≤‡∏™‡∏ï‡∏£‡πå)

```python
# Difference (‡∏™‡πà‡∏ß‡∏ô‡∏ï‡πà‡∏≤‡∏á) - ‡∏Ñ‡πà‡∏≤‡∏ó‡∏µ‡πà‡∏≠‡∏¢‡∏π‡πà‡πÉ‡∏ô a ‡πÅ‡∏ï‡πà‡πÑ‡∏°‡πà‡∏≠‡∏¢‡∏π‡πà‡πÉ‡∏ô b

# Symmetric Difference (‡πÑ‡∏°‡πà‡∏ã‡πâ‡∏≥‡∏Å‡∏±‡∏ô) - ‡∏Ñ‡πà‡∏≤‡∏ó‡∏µ‡πà‡∏≠‡∏¢‡∏π‡πà‡πÉ‡∏ô a ‡∏´‡∏£‡∏∑‡∏≠ b ‡πÅ‡∏ï‡πà‡πÑ‡∏°‡πà‡∏ó‡∏±‡πâ‡∏á‡∏Ñ‡∏π‡πà

# Subset/Superset

{1, 2}.isdisjoint({3, 4})           # True (‡πÑ‡∏°‡πà‡∏°‡∏µ‡∏™‡∏°‡∏≤‡∏ä‡∏¥‡∏Å‡∏£‡πà‡∏ß‡∏°)
```

### Packing/Unpacking ‡πÅ‡∏•‡∏∞ Starred Unpacking
```python
# Packing
pt = 13.0, 5.5          # (13.0, 5.5)

# Unpacking
x, y = pt               # x=13.0, y=5.5

# ‡∏™‡∏•‡∏±‡∏ö‡∏Ñ‡πà‡∏≤

a, b = b, a             # a=2, b=1

# Starred

*head, tail = (10, 20, 30)            # head=[10,20], tail=30
```
